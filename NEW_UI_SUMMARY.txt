┌──────────────────────────────────────────────────────────────────────────────┐
│                         💡 CUSTOMIZATION TIPS                                │
└──────────────────────────────────────────────────────────────────────────────┘

Change Colors:
  • Edit app/src/main/res/values/colors.xml
  • Update primary, accent, and text colors
  • Restart app to see changes

Adjust Animations:
  • Open MainActivity.kt, ImageProcessingActivity.kt, ResultActivity.kt
  • Find .animate() calls
  • Modify .setDuration() values
  • Change .setStartDelay() timing

Modify Text:
  • Edit app/src/main/res/values/strings.xml
  • Update any UI text
  • No code changes needed

Change Layout:
  • Edit XML files in app/src/main/res/layout/
  • Modify padding, margins, sizes
  • Use Android Studio's Layout Editor
