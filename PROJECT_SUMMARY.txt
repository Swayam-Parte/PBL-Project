  🎯 WHAT THE APP DOES
═══════════════════════════════════════════════════════════════════════════════

USER WORKFLOW:
1. Launch app
2. Select image source (Gallery 📁 or Camera 📷)
3. Select/capture tooth image
4. Tap "Analyze Shade" button
5. View results:
   • Predicted shade class (A/B/C/D)
   • Confidence percentage
   • Probability distribution for all classes
6. Option to analyze another image

TECHNICAL WORKFLOW:
1. MainActivity loads ONNX model from assets on startup
2. User selects Bitmap image
3. AndroidStandardScaler preprocesses image:
   - Resize to 224×224
   - Normalize to [0, 1]
   - Apply ImageNet standardization
   - Convert to NCHW tensor format
4. OnnxModelInference runs prediction:
   - Create ONNX tensor
   - Execute inference
   - Apply softmax
   - Format results
5. UI displays results with visual feedback

  🔍 TECHNICAL SPECIFICATIONS
═══════════════════════════════════════════════════════════════════════════════

Model:
  • Format: ONNX (Random Forest)
  • Location: app/src/main/assets/tooth_shade_model.onnx
  • Input Shape: [1, 3, 224, 224] (NCHW)
  • Output: 4 class probabilities (A, B, C, D)

Dependencies:
  • ONNX Runtime Android: 1.17.0
  • Coil Compose: 2.5.0
  • Jetpack Compose BOM: Latest
  • Material 3: Latest

Android Requirements:
  • Min SDK: 24 (Android 7.0)
  • Target SDK: 35 (Android 15)
  • Compile SDK: 35
  • Java Version: 11

Preprocessing:
  • Input Size: Any size → 224×224
  • Normalization: [0, 1] range
  • Standardization: ImageNet (mean=[0.485,0.456,0.406], std=[0.229,0.224,0.225])
  • Format: NCHW (Channels first)

Performance:
  • Model Load: 1-3 seconds (first launch)
  • Inference Time: 0.5-2 seconds
  • Memory Usage: ~150-300 MB
  • APK Size: ~20-30 MB
